The parameters.xml and context.xml files must be placed in the FakeNewsSpreadSim.rs folder and the batch_params.xml file in the batch folder, within the Repast Simphony FakeNewsSpreadSim project directory tree.

.
├── FakeNewsSpreadSim
         ├── FakeNewsSpreadSim.rs
         │       ├── parameters.xml
         │       └── context.xml
         └── batch
                 └── batch_params.xml
