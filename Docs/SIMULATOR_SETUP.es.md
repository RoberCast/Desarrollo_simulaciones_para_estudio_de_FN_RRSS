# Configuración del simulador
Después de crear el proyecto de Repast Simphony y tras colocar los archivos indicados en el [README.es.md](https://github.com/RoberCast/Desarrollo_simulaciones_para_estudio_de_FN_RRSS/blob/main/README.es.md#instrucciones) en su lugar correspondiente, se procede a completar la configuración del simulador. Para ello, se realizan los pasos siguientes:

* **Ejecución del simulador en modo GUI (ejecución única)**. En el IDE Eclipse que se ha instalado con Repast Simphony, se ejecuta *FakeNewsSpreadSim Model*. Se abrirá el simulador en modo GUI y para proceder a su configuración.
  
<p align="center">
  <img src="../Images/Simulator_setup/EclipseRun.png" alt="FakeNewsSpreadSim Model." width="80%"/>
</p>

* **Pestaña Scenario Tree**: Con el simulador abierto, se pincha en la pestaña *Scenario Tree*, que contiene la configuración de la simulación.
